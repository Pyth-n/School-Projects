Object Creations

With the new keyword

		var x = new Object();
		x.num = 20;
		x.age = 22;

With the object literal

This is the preferred way

		var x = {num:20, age:22};

Properties

Properties are the values that an object holds.

An example is the code in bold

		// With an object literal
		var cellphone = {model:"note", generation:8};

		// With a new object
		var cellphone = new Object();
		cellphone.model = "note";
		cellphone.generation = 8;

Methods

A method is the behavior of an object

		var dog = new Object();
		// This is considered a method for the dog object
		dog.bark = function(){
		console.log("woof");
		}

Instantiation

Instantiation is the initialization of an object

If we have class, we can instantiate it later on and use its properties/methods

		function Person(){
			age = 21;
			eyeColor = "Brown";
		}

		// This instantiates a Person object
		var x = new Person(){
			console.log("Brown");
		}

Inheritance

This is when an object shares its properties with another object

Here, we see that the base Animal class gives its properties to the Dog class

		function Animal() = {}; // Base class Type
		// This gives its __proto__ object parent the function. Memory efficient
		Animal.prototype.eats = function(){
			console.log("All animals eat");
		}

		
		function Dog() {}; // Sub class Type
		// This inherits the Animal's properties to the Dog class
		Dog.prototype = new Animal();
		Dog.prototype.bark = function(){
			return "WOOF";
		}

		var doggo = new Dog();
		document.getElementById("dog").innerHTML = doggo.bark();

If we instantiate and call the methods, we'll see that Dog inherited Animal's properties
