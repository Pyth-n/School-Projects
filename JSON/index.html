JSON

JSON stands for JavaScript Object Notation

Its purpose is to store and exchange data in a specific format.

This is used because the data that is exchanged can only be text.

It is also common to stringify objects to store in APIs such as local storage

Example: Stringify

An object can be converted into JSON with the stringify function

// A Person object
var person = {"name":"david", "age": 23, "birthMonth":"December"};

// Convert to JSON
var JSONed = JSON.stringify(person);

Example: Parse

Parsing is converting JSON text into a JavaScript object

// Continuing from the previous code
var toParse = JSON.parse(JSONed);
