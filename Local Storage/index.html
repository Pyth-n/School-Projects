Local Storage

It is an object that store an object with no expiration date. Won't be deleted when it's closed.

Values are accessed by keys. They can be inserted, retreived, and deleted

Syntax

Store Data:
localStorage.setItem(KEY, VALUE);

Retrieve Data:
localStorage.getItem(KEY);

Remove Data:
localStorage.removeItem(KEY);

Example:

// Store, retrieve, and remove a lastname into localStorage
localStorage.setItem("lastName", "Perez");

localStorage.getItem("lastName");

localStorage.removeItem("lastName");
